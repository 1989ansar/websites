    • Projects

    This is where we keep a collection of all the projects that people are working on using ev3dev. We invite you to click through the links below to see what cool stuff ev3dev can do!

    Would you like your project added to the list? We would! Create a page and send us a pull request.

    • Ev3 Print3rbot

      An EV3 robot that draws with a pen. Can draw directly from SVG images, or with a mouse.

    • Dalek

      The Daleks are evil aliens from Doctor Who, and are the Doctor’s most fearsome enemy. Inside the armoured...

    • Snake Game

      A simple snake game for Lego EV3 running ev3dev firmware.

    • Ev3 Controlled Rcx

      Remotely control an RCX using your EV3 and a USB IR tower!

    • Wiimote Controlled Rover

      Drive your EV3 using a Nintendo Wiimote over bluetooth!

    • Treev3

      A web interface for turning on/off Christmas trees using the Dexter Industries dSwitch with ev3dev

    • Ros On Ev3

      Robot Operating System (ROS) on ev3dev.

    • Led Demo

      Starting with kernel version 3.16.1-1-ev3dev, the brightness attribute now works for the LEDs.

    • Ev3 Simple Goodies

      Helpful Bash scripts that test and demonstrate cool ev3dev functionality.

    • Balancing Robot

      This is still a work in progress. It is a Segway-style balancing robot based on the HTWay. You can control it with the EV3 IR Remote/Sensor.

    • Python Rubiks Cube Solver

      The model is based on mindcub3r, from David Gilday.

    • Example Project

      An example/template project to make creating a new project page easier.
