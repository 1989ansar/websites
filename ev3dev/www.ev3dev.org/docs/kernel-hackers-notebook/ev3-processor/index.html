    Essentials

    • Texas Instruments AM1808 ARM Microporcessor
    • 32-bit
    • ARM9
    • All internal pullup/pulldown resistors are disabled

    Documentation

    • Official Website
    • AM1808 ARM® Microprocessor Datasheet (pdf)
    • AM1808 Technical Reference Manual (pdf)
    • AM1808 Pinmux Utility (pdf)

    System Components

    Used by EV3

    • 3 - 16550-Type UART Modules
    • 2 - Serial Peripheral Interfaces (SPI)
    • 2 - Multimedia Card (MMC)/Secure Digital (SD) Card Interfaces (using one)
    • 2 - Master/Slave I2C Interfaces (using one)
    • Programmable Real-Time Unit Subsystem (PRUSS)
    • USB 1.1 Host Port (OHCI)
    • USB 2.0 On-The-Go Port (OTG)
    • Real-Time Clock (RTC)
    • 1 - 64-bit General-Purpose or Watchdog Timer
    • 2 - Enhanced High-Resolution Pulse Width Modulators (eHRPWM)
    • 3 - 32-bit Enhanced Capture Modules (ePCAP)

    Not used by EV3

    • LCD Controller (EV3 uses an external controller)
    • 1 - Host Port Interface (HPI)
    • 1 - Multichannel Audio Serial Port (EV3 uses PWM for sound)
    • 2 - Multichannel Buffered Serial Ports
    • 10/100 Mbps Ethernet MAC (EMAC)
    • Video Port Interface (VPIF)
    • Universal Parallel Port (uPP)
    • Serial ATA (SATA) Controller

    Not sure yet

    • Enhanced Direct Memory Access Controller 3 (EDMA3)
    • 3 - 64-bit General-Purpose Timers

    Pin Assignments

    There are 400 pins on the chip. The pins are multiplexed so that each pin can perform multiple functions. This is how the EV3 uses each pin. See section 10.4.9 in the AM1808 technical reference manual to see where this came from.
