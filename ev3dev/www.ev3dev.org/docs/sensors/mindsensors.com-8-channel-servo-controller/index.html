    # Modes 2

    Modes

    mode Description units decimals3 num_values Values
    V3 EV3 Compatible V (volts) 3 1 value0: Battery voltage (0 to 9400)4
    OLD5 Older versions V (volts) 3 1 value0: Battery voltage (0 to 9400)4

    Commands

    This sensor does not support commands.

    Notes

    1. The address is programmable. See manufacturer documentation for more information. ↩

    2. The ms-8ch-servo driver loads separate servo motor devices (one for each of the 8 channels) in addition to the lego-sensor class device. See the servo-motor class for more information. The servo-motor class port_name attribute will return in<N>:sv<M> where <N> is the input port the servo controller is connected to and <M> is the channel as indicated on the servo controller itself. ↩

    3. Decimal places. For example, if the range of a value is from 0 to 1000 and decimals is 1, then the acutal range is 0.0 to 100.0 in the units specified. ↩

    4. The current voltage scaling is based on the manufacturers documentation, however it seems to be low. If you are seeing this too, please open an issue on GitHub and we will change the scaling. ↩ ↩2

    5. Older versions of this sensor have the battery voltage at a different address. If the default mode does not return a value, try this mode. ↩
