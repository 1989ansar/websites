    General Info

    device_name ms-absolute-imu
    Website www.mindsensors.com
    Connection I2C/NXT
    Address 0x111
    sysfs class lego-sensor
    vendor_id mndsnsrs
    product_id AbsIMU
    # Modes 6

    Modes

    mode Description units decimals2 num_values Values
    TILT Tilt deg (degrees) 0 3 value0: X-axis angle (0 to 180)
    value1: Y-axis angle (0 to 180)
    value2: Y-axis angle (0 to 180)
    ACCEL3 Acceleration g (Standard gravity) 0 3 value0: X-axis acceleration
    value1: Y-axis acceleration
    value2: Z-axis acceleration
    COMPASS4 Compass deg (degrees) 0 1 value0: Heading (0 to 360)
    MAG4 Magnetic field none 0 3 value0: X-axis magnetic field
    value1: Y-axis magnetic field
    value2: Z-axis magnetic field
    GYRO5 Gyro d/s (degrees per second) 1 3 value0: X-axis rotational speed
    value1: Y-axis rotational speed
    value2: Z-axis rotational speed
    ALL6 All data none 0 23

    Commands

    command Description
    BEGIN-COMP-CAL Begin compass calibration
    END-COMP-CAL End compass calibration
    ACCEL-2G7 Change accelerometer sensitivity to 2G and gyro sensitivity to 250 deg/sec
    ACCEL-4G7 Change accelerometer sensitivity to 4G and gyro sensitivity to 500 deg/sec
    ACCEL-8G7 Change accelerometer sensitivity to 8G and gyro sensitivity to 2000 deg/sec
    ACCEL-16G7 Change accelerometer sensitivity to 16G and gyro sensitivity to 2000 deg/sec

    Notes

    1. The address is programmable. See manufacturer documentation for more information. ↩

    2. Decimal places. For example, if the range of a value is from 0 to 1000 and decimals is 1, then the acutal range is 0.0 to 100.0 in the units specified. ↩

    3. Only returns data from models with an accelerometer (AbsoluteIMU-AC/AbsoluteIMU-A). ↩

    4. Only returns data from models with a compass (AbsoluteIMU-C/AbsoluteIMU-AC/AbsoluteIMU-ACG). ↩ ↩2

    5. Only returns data from models with a gyro (AbsoluteIMU-ACG). ↩

    6. Reads all data from the sensor. Use bin_data attribute to read values. Some values will not be scaled. See manufacturer docs for more info. ↩

    7. Wait 50 msec after sending command for sensor to reconfigure itself. ↩ ↩2 ↩3 ↩4
