    General Info

    device_name ht-nxt-eopd
    Website www.hitechnic.com
    Connection Analog/NXT
    sysfs class lego-sensor
    # Modes 2

    Modes

    mode Description units decimals1 num_values Values
    LONG Proximity (long range) none 0 1 value0: Proximity (0-100)2
    SHORT Proximity (short range) none 0 1 value0: Proximity (0-100)2

    Commands

    This sensor does not support commands.

    Notes

    1. Decimal places. For example, if the range of a value is from 0 to 1000 and decimals is 1, then the acutal range is 0.0 to 100.0 in the units specified. ↩

    2. This value is the square root of the raw value. You can derive a value proportional (linear) to distance by dividing a constant by this value, e.g. 35 / value0. ↩ ↩2
