Solution 18.1. A Point class

Exercise 18.1

class Point:

    def __init__(self, x, y):
        self.x = x
        self.y = y

    def move(self, dx, dy):
        self.x = self.x + dx
        self.y = self.y + dy

    def show(self):
        return self.x, self.y
